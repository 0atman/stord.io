How it works

To create a new item, POST mykey=hello to /key/.


To update it, PUT value=new_value to /key/mykey.


That's it!

http verb params endpoint desription
POST mykey=hello /key/ create a new item
GET /key/mykey get an item
PUT value=new_value /key/mykey update an item
