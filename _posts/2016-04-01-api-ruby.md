### POST

```ruby
require 'net/http'

uri = URI('stord.io/key?auth=1234')
res = Net::HTTP.post_form(uri, 'hello' => 'world')
puts res.body
=> '{"hello": "world"}'
```

### GET

```ruby
uri = URI('stord.io/key/hello?auth=1234')
puts Net::HTTP.get(uri)
=> '{"hello": "world"}'
```
